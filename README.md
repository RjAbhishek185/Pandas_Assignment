# Basic Data Exploration and Cleaning using Pandas

## Objective
Learn Python basics and perform data exploration and cleaning using the Pandas library.

## Dataset
Shopping Dataset from Kaggle:
https://www.kaggle.com/datasets/anvitkumar/shopping-dataset

## Tasks Performed

1. Loaded the CSV dataset into a Pandas DataFrame.
2. Explored the dataset using:
   - head()
   - tail()
   - shape
   - columns
   - dtypes
3. Identified and handled missing values.
4. Performed data filtering and column selection.
5. Removed duplicate records.
6. Created a derived column:
   - total_amount = final_price × quantity
7. Saved the cleaned dataset as a new CSV file.

## Files Included

- Data_Exploration_Cleaning.ipynb
- cleaned_shopping_dataset.csv

## Tools Used

- Python
- Pandas
- NumPy
- Jupyter Notebook
- GitHub

## Outcome

The dataset was successfully explored, cleaned, transformed, and saved for further analysis.
